## 개인 프로젝트
# ARS (Algorithm Review Service)
알고리즘 복습을 편하게 할 수 있는 "저"만의 웹서비스입니다. <br>
등록된 문제에 계속해서 복습 내용을 추가할 수 있고, 문제 상세 보기에 복습 목록이 함께 보여집니다. <br>
복습 알림 서비스 기능으로 알림 날짜를 설정하면 해당 날짜에 Slack을 통해 복습 알림을 받을 수 있습니다.
